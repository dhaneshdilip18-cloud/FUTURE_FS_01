# Dhanesh Kambala — Portfolio Website

> A modern, animated personal portfolio built with React, TypeScript, and Framer Motion.

🌐 **Live Site:** [dhanesh-portfolio-snowy.vercel.app](https://dhanesh-portfolio-snowy.vercel.app/)

---

## Overview

This is the source code for Dhanesh Kambala's personal portfolio — a full-stack developer and B.Tech CSE student at Miracle Educational Society Group of Institutions. The portfolio showcases skills, projects, education, certifications, and contact information through a visually rich, animated single-page application.

---

## Features

- **Animated Hero Section** — Typewriter effect, floating icons, gradient profile avatar, and social links
- **About Me** — Personal introduction with trait cards and stat highlights
- **Skills** — Categorized skill cards (Frontend, Backend, Database, Languages, Tools) with animated progress bars
- **Projects** — Filterable project cards with tech tags, feature lists, GitHub and demo links
- **Education Timeline** — Alternating vertical timeline for academic history
- **Certifications** — Sticky-note style certification cards
- **Achievements** — Icon-driven achievement cards
- **Resume Download** — Direct `.docx` resume download
- **Contact Form** — Functional contact form with submission feedback
- **Dark / Light Mode** — Toggle with persistent state
- **Scroll Progress Bar** — Top progress indicator
- **Floating Particles Background** — Ambient animated particles
- **Mobile Responsive** — Fully responsive with a hamburger navigation menu
- **Back to Top Button** — Fixed floating button

## Getting Started

### Prerequisites

- Node.js v18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/dhaneshdilip18-cloud/<repo-name>.git
cd FUTURE_FS_01-main

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173`.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

### Type Check

```bash
npm run typecheck
```
